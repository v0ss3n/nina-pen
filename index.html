Nina Pen

What would make the green more green? Should I add some yellow, blue or white?

If I add some black, it would be in balance again. I don’t like balance. Let me add some yellow, to make it more daring.

What if the paint would have a structure? A leathery like fabric. The sound of cutting this fabric is so fulfilling. I could cut and create any shape as if the fabric is like a statue. I like statues.

Now add dots, let’s say 200 of them. Every dot should have an even smaller dot of white paint. Not blue or green, but white. Yes, white would be good.

I made 100 drawings, all in black and white. Every line should be straight, no curves, no hesitation. Continue.

Green, pink, green, pink, blue.
